# Tables in Markdown

Syntax:

    |<COLUMN_NAME_1> |<COLUMN_NAME_2>|
    |:--------------:|--------------:|
    |<1x1>           |<1x2>          |
    |<2x1>           |<2x2>          |

<br>

    Note: 
             '|:--|'  => Left Aligned Table
            '|:---:|' => Centered Table
             '|--:|'  => Right Aligned Table

## Cat Table:

|Cat Name| Cat Color|
